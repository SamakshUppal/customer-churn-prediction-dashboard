# Customer Churn Prediction Dashboard

This project builds a machine learning model to predict telecom customer churn and provides an interactive dashboard using Streamlit.

## Project Overview

Customer churn is a major problem for telecom companies. This project analyzes customer behavior and predicts whether a customer is likely to leave the service.

The project demonstrates an end-to-end machine learning pipeline including data analysis, model training, and deployment through a web dashboard.

## Dashboard Preview

![Dashboard](images/dashboard.png)

## Features

- Exploratory Data Analysis (EDA)
- Machine Learning Model (Random Forest)
- Data preprocessing and feature scaling
- Interactive prediction dashboard using Streamlit
- Visualization of churn patterns

## Tech Stack

Python  
Pandas  
Scikit-learn  
Streamlit  
Matplotlib  
Seaborn

## Dataset

IBM Telco Customer Churn Dataset

The dataset contains customer information such as tenure, billing charges, services used, and churn status.

## Model

The project uses a Random Forest classifier to predict customer churn.

Features used in the prediction model:

- Tenure
- MonthlyCharges
- TotalCharges

## How to Run the Project

1. Clone the repository

2. Install required libraries

3. Run the Streamlit dashboard

## Dashboard

The Streamlit dashboard allows users to enter customer information and predict whether the customer is likely to churn.

## Project Pipeline

Data Collection → Data Cleaning → Exploratory Data Analysis → Model Training → Model Deployment → Interactive Dashboard
